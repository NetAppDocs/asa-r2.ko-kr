== 보안

[cols="2,4"]
|===
| 업데이트 | 설명 


| link:../secure-data/enable-anti-ransomware-protection.html["SnapMirror 활성 동기화 관계를 통한 ARP/AI 지원"] | 인공지능 기반 자율 랜섬웨어 보호(ARP/AI)는 ASA r2 시스템의 SnapMirror 활성 동기화 관계에 있는 기본 읽기/쓰기 스토리지 유닛을 지원합니다. ARP/AI는 활성 기본 스토리지 유닛에서 실행되어 기본 전용 ARP 스냅샷을 생성하고 해당 스토리지 유닛에 대한 머신 러닝 기반 랜섬웨어 탐지 기능을 제공합니다. 


| link:../learn-more/additional-supported-ontap-features.html#tls-hardware-offload["TLS 하드웨어 오프로드 지원"] | 지원되는 이더넷 카드로 전체 클러스터의 전송 계층 보안(TLS) 암호화 및 복호화를 오프로드할 수 있습니다. 이를 통해 CPU 오버헤드가 줄어들고 TLS를 사용하는 시스템의 성능이 향상됩니다. 
|===


== 스토리지 효율성

[cols="2,4"]
|===
| 업데이트 | 설명 


| link:../learn-more/additional-supported-ontap-features.html#global-deduplication["전역 중복 제거 지원"] | 글로벌 데이터 중복 제거는 기존 ONTAP 데이터 중복 제거를 통해 확보되는 공간 절약 효과를 전체 Storage Availability Zone(SAZ)으로 확장합니다. SAZ 내 모든 스토리지 장치에서 중복 데이터 블록을 제거함으로써 스토리지 효율성을 크게 향상시키고 총 소유 비용을 절감할 수 있습니다. 글로벌 데이터 중복 제거는 기본적으로 활성화되어 있으며 관리자 구성 없이 투명하게 작동합니다. 
|===
